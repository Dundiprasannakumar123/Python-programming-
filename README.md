PROBLEM STATMENT-2
WRITE A PROGRAM TO FIND THE NUMBER OF DIGITS IN A GIVEN NUMBER.
num=int(input())
count=0
i = num
while(i>0):
    count=count+1
    i=i//10
print(f"The number of digits in {num}:",count)
